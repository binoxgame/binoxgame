# 🎮 Binox Game

**Binox Game** is a blockchain-based play-to-earn gaming platform built on **BNB Smart Chain (BSC)**, part of the **BNB Chain ecosystem**.

Players can participate in multiple mini-games and earn **BNX tokens** and reward points instantly. The platform integrates **BNB payments, smart contracts, and a React frontend** to provide a fast, transparent, and decentralized gaming experience.

🌐 Website: https://binoxgame.fun  

---

# 🚀 Technology Stack

- **Blockchain:** BNB Smart Chain (BSC) — part of the BNB Chain ecosystem  
- **Smart Contracts:** Solidity ^0.8.20  
- **Frontend:** React.js  
- **Web3 Integration:** ethers.js  
- **Oracle:** Chainlink BNB/USD Price Feed  
- **Development:** Hardhat + OpenZeppelin

# ⚡ Core Features

### 🎮 Play-to-Earn Games
Users can play multiple mini-games including:

- Spin the Wheel  
- Balloon Pop  
- Crypto Mines  

Players earn **BNX tokens and reward points** instantly.

---

### 💰 BNB Payment Gateway

Users can deposit using **BNB** through a secure smart contract payment gateway.

Payments are validated using **Chainlink BNB/USD price feeds**.

---

### 📊 Real-Time Price Conversion

The contract converts **BNB → USD value** using **Chainlink oracle feeds** to enforce payment limits.

---

### 🔒 Secure Smart Contract

Security mechanisms include:

- Chainlink price oracle
- Minimum & maximum USD limits
- Owner-controlled payment receiver
- Secure BNB transfers
